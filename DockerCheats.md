##### run an image
```shell
docker run -p [port]:[port] [image]
```

##### build an image
```shell
docker build .
```
##### pull an image
```shell
docker pull [nginx, etc]
```

##### stop a container
```shell
docker stop [name]
```
##### list containers
```shell
docker ps
```
##### list all containers
```shell
docker ps -a
```
